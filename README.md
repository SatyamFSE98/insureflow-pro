# InsureFlow Pro

Enterprise-level Health Insurance Management Platform built using Angular, Java, Spring Boot Microservices, MySQL, MongoDB, Kafka, Redis, Docker, Jenkins, and Cloud deployment.

## Main Features

- User registration and login
- Policy management
- Claim management
- Payment simulation
- Kafka-based notification system
- Redis caching
- MongoDB audit logging
- Chatbot for user queries
- Swagger/OpenAPI documentation
- Dockerized deployment
- Jenkins CI/CD pipeline

## Architecture

Angular Frontend -> API Gateway -> Spring Boot Microservices -> MySQL/MongoDB/Kafka/Redis

## Services

- API Gateway
- Discovery Server
- Config Server
- User Service
- Policy Service
- Claim Service
- Payment Service
- Notification Service
- Audit Service
- Chatbot Service
- Report Service

## Goal

This project is built for enterprise-level full-stack learning and interview preparation.
