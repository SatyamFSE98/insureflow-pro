# Phase 0 - Project Overview

## Project Name

InsureFlow Pro

## Domain

Health Insurance Management System

## Architecture Style

Microservices Architecture

## Frontend

Angular will be used to build clean user interfaces for customers, policies, claims, payments, reports, and chatbot.

## Backend

Spring Boot microservices will handle business logic.

## Databases

- MySQL for transactional data
- MongoDB for audit logs and chatbot history

## Messaging

Kafka will be used for asynchronous communication between services.

## Cache

Redis will be used for frequently accessed dashboard and policy data.

## Deployment

Docker and Jenkins will be used for CI/CD and deployment.
